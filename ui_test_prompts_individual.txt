# Mini-Nebulus Individual Feature Test Prompts

Copy and paste these prompts one by one to verify each feature in isolation.

## 1. Task Management
**Prompt:**
"Create a plan to learn about this project. Add a task to read the context file."

*Expected Action:* creates a plan, adds a task.

## 2. Context Management
**Prompt:**
"Pin the file 'CONTEXT.md' to the context and then list the current context."

*Expected Action:* calls `pin_file`, then `list_context`.

## 3. File Operations
**Prompt:**
"List the files in the 'mini_nebulus' directory using a shell command."

*Expected Action:* calls `run_shell_command` with `ls -F mini_nebulus` (or similar).

## 4. Skill Execution
**Prompt:**
"Use the file_info skill to check the size of 'mini_nebulus/main.py'."

*Expected Action:* calls `file_info` skill.

## 5. RAG / Semantic Search
**Prompt:**
"Search the codebase for 'AgentController' and tell me which file it is in."

*Expected Action:* calls `search_code`.

## 6. File I/O
**Prompt:**
"Write a file named 'test_hello.txt' with the content 'Hello World'."

*Expected Action:* calls `write_file`.
